## Run Locally

Clone the project

```bash
  git clone https://github.com/ErtanMuslic/NineMensMorris-Backend.git
```

Go to the project directory

```bash
  cd NineMensMorris-Backend
```

Install dependencies

```bash
  pip install
```

Go to the folder that contains manage.py

```bash
  cd ninemanmorris
```

Start the server

```bash
  py .\manage.py runserver
```
